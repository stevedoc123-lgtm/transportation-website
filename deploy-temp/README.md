# Film Transportation Services Website

## Overview
Professional website with run request form for film production transportation services.

## Features
- Clean, mobile-responsive design
- Netlify Forms integration (free tier: 100 submissions/month)
- Form notifications to: stevedoc123@gmail.com
- Success page after submission
- Anti-spam honeypot protection

## Form Fields
All fields from original JotForm replicated:
- Requestor name, department, phone, email
- Request type (pickup, delivery, exchange, return)
- Dates and times (of run, needed)
- Payment options
- Pickup contact and address
- Delivery contact, phone, and address
- Vehicle type needed
- Item description
- Special instructions

## Deployment Instructions

### Step 1: Create Netlify Account
1. Go to https://netlify.com
2. Sign up with GitHub or email
3. Verify email

### Step 2: Deploy Site
**Option A: Drag & Drop (Easiest)**
1. Zip this folder (website-transportation)
2. Go to Netlify dashboard
3. Drag zip file to deploy area
4. Site goes live instantly

**Option B: Git Integration (Recommended for updates)**
1. Push this folder to GitHub repo
2. Connect repo in Netlify
3. Auto-deploy on every push

### Step 3: Configure Form Notifications
1. In Netlify dashboard, go to Forms
2. Click on "run-request" form
3. Add notification: stevedoc123@gmail.com
4. Test with a submission

### Step 4: Custom Domain (Optional)
1. Buy domain (Namecheap, Google Domains, etc.)
2. In Netlify: Domain settings → Add custom domain
3. Update DNS records as instructed
4. Enable HTTPS (free SSL)

## Cost
